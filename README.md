# Car Price Prediction 

##  Overview
This project builds a regression model to predict car prices based on different features.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

##  Task Details
- Data preprocessing and cleaning
- Feature engineering (e.g., car age)
- Built regression model
- Evaluated using MAE and R² score

##  Result
Model successfully predicts car prices with reasonable accuracy.
